
# My Game List

A web application for tracking and managing your favorite video games.

## Features

- Add games to your personal list (WIP)
- Organize games by genre, platform, or status (WIP)
- Search and filter functionality (WIP)
- Responsive design for desktop and mobile


## Tech Stack

- **Frontend Framework:** React
- **Languages:** JavaScript, HTML, CSS


## Getting Started

### Installation

```bash
npm install
```

### Running Locally

```bash
npm start
```

The application will be available at localhost.
